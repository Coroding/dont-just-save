# Asset List

## Current Version Uses Generated HTML/CSS Assets

- Abstract source cards for social post, video note, and web article.
- Placeholder Android phone frame and app screens.
- CSS-only oil-paint color patches using soft blurred shapes.
- CSS-only topic cards, status chips, buttons, source labels, and logo mark.
- No external images or platform logos.

## Copyright And Brand Safety

- 小红书, B站, 微信, and other real platform logos are not used.
- Source platforms are represented by generic labels: 社交笔记, 视频内容, 网页文章, 公众号灵感.
- UI screens are product mockups created for the demo rather than screenshots from third-party products.

## Future Optional Assets

- Real app screenshots once the Android UI is ready.
- A custom app icon export from the Android project.
- A soft paper texture bitmap if a more tactile hand-account feel is needed.
- Light background music or subtle UI sounds, if the portfolio context supports audio.
